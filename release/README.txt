SQUEEPTY 2600 Demo
==================

A compact Atari 2600 demo for NTSC systems.

Release date: 20 June 2026
Production: SQUEEPTY 2600 Demo
Platform: Atari 2600 / Atari VCS (NTSC)
Type: demo
Group: Squeepty

FILES
-----

SQUEEPTY.BIN    4096-byte, plain 4K Atari 2600 cartridge ROM
SQUEEPTY.NFO    Release information and credits
FILE_ID.DIZ     Short file description

RUNNING
-------

Open SQUEEPTY.BIN in an Atari 2600 emulator such as Stella, or use an
appropriate 4K-compatible flash cartridge on NTSC Atari 2600 hardware.

The demo is automatic; it does not use any controls.

FEATURES
--------

* Color-cycling raster bars and title background
* Large SQUEEPTY playfield title
* Bouncing player sprite
* Animated checker playfield
* Scrolling GREETINGS TO ALL ATARI DREAMERS... ticker
* Two-voice TIA melody
* Fixed 262-scanline NTSC frame

TECHNICAL
---------

The ROM uses no bank switching and is exactly 4,096 bytes.
